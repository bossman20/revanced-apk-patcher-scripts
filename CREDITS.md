## What This Repository Provides

This repository **ONLY** provides:

- Windows batch scripts (`.bat`) for easy patching
- PowerShell scripts (`.ps1`) for interactive patch selection
- Documentation and setup guides

**We do NOT provide**:

- The ReVanced CLI (you download it from official sources)
- The patch bundles (you download them from official sources)
- The base APK files (you must obtain these legally)
